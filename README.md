# AE Nails & Lashes Siedlce

Statyczna strona salonu AE Nails & Lashes w Siedlcach z portfolio, pełnym cennikiem i rezerwacją online.

## Pliki

- `index.html` - struktura strony
- `styles.css` - oprawa wizualna i responsywność
- `script.js` - galeria, lightbox, wyszukiwarka i filtrowanie cennika
- `assets/` - zdjęcia portfolio, tatuaży i zespołu
